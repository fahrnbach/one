# Overview

This is the canonical SparkDocs generation template. It standardizes the documentation process across the Fahrnbach universe by guiding iterative AI-generated content creation using structured YAML frontmatter, markdown content, and meta-cognitive scaffolding.
```
[[[

Intent: To iteratively generate and refine documentation for the entire fahrnbach.one universe.

Process:
- **PreCog (Pre-process):** Have a conversation with ChatGPT and uncover best practices, missing links, and relevant inspirations.
  See more at `/meta/overview/pre-cog`
- **Generate:** Use this template and context to generate a living draft.
- **Score:** Human should Read the Ai generated doc and assign manual scores for `alignment` and `confidence`.
- **MetaCog:** Re-read, refine, and regenerate sections based on evaluation.
- **Repeat:** Iterate as needed, incrementing the `revision` number and updating the `updated` date.

]]]
```

## Usage
Pass this template to your Ai and ask it to render documents with it.

```md
<!-- Prompt: Generate canonical template article for SparkDocs project -->
<!-- AI Notes: Use this template. Fill in content inside [[[]]] block. -->
```
